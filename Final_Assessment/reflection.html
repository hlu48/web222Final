Explain the concept of Responsive Design

Responsive design means the website can automatically adjust its windows size on different devices, e.g. Phone, PC, TV. If user is using a smartphone, the website should be able to adjust its resolution, screen size and orientation on its own, so user would not need to manually zoom in or scroll horizontally. Developers should always implement responsive design approach and test on various of systems to improve user experiences and make management easier.

Responsive Design & Web Technologies

Responsive design approach can be applied on various web technologies. Responsive design is not just about changing screen resolution or orientation, it consists of a mix of flexible grids and layouts, images and an intelligent use of CSS media queries. The fluid grid concept calls for page element sizing to be in relative units like percentages, rather than absolute units like pixels or points. Flexible images are also sized in relative units, so as to prevent them from displaying outside their containing element. Responsive design greatly improves web technologies performance, and reduces cost of maintenance for web technologies.
